# Indipa Ayomal — Portfolio

Personal portfolio website built with React, TypeScript, Vite, shadcn-ui, and Tailwind CSS.

## Getting started

Requirements: Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

This project is deployed on [Netlify](https://www.netlify.com). To deploy your own copy, connect the repository to Netlify and set the build command to `npm run build` with the publish directory set to `dist`.
